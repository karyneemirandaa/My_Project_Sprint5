# Análise de Veículos Usados

## Descrição do Projeto

Este é um aplicativo web desenvolvido com **Streamlit** que permite analisar dados de veículos usados através de visualizações interativas.

## Funcionalidades

O aplicativo oferece as seguintes funcionalidades:

- **Histograma interativo**: Visualiza a distribuição da quilometragem dos veículos
- **Gráfico de dispersão**: Analisa a relação entre quilometragem e preço dos carros
- **Interface intuitiva**: Botões simples para gerar diferentes tipos de gráficos
- **Visualizações responsivas**: Gráficos que se adaptam ao tamanho da tela

## Como usar

1. Execute o aplicativo: `streamlit run app.py`
2. Clique no botão "Criar histograma" para ver a distribuição da quilometragem
3. Clique no botão "Criar gráfico de dispersão" para analisar preço vs quilometragem

## Tecnologias utilizadas

- **Python**
- **Streamlit** - Framework para aplicações web
- **Pandas** - Manipulação de dados
- **Plotly Express** - Visualizações interativas
